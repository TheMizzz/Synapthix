# Synapthix — Литературный Ассистент

Synapthix — это локальный помощник по книгам, работающий на вашем компьютере. Он умеет:

- Искать книги по описанию (через Google Books API)
- Загружать ваши книги (.txt)
- Искать фрагменты текста по смыслу
- Отвечать на вопросы по содержанию
- Давать литературные рекомендации

## Установка

### Примечание

Вы должны убедится, что на диске C: (если windows) или на диске / (если linux) есть свободная **память в размере 5Gb**

### Windows
1. Зайдите в cmd и пропишите git clone ttps://github.com/TheMizzz/Synapthix

2. Запустите `launch_synapthix.bat` **от имени администратора**.

3. Следуйте инструкциям на экране.


## Запуск

После установки запустится **FastAPI-сервер**. Откройте в браузере:

`http://localhost:8000/static/index.html`

## Совместимость

- Windows 10/11 (64-bit)
- Linux (Ubuntu, CentOS, Fedora, Arch и т.д.)
- macOS (требуется ручная установка Ollama)

## Технологии

### Технологии для ИИ

- Ollama + qwen3-vl:4b

- LangChain + FAISS (RAG)
- Google Books API

### Языки программирования

- **Python**
- **JavaScript**
- **HTML** + **CSS**

## Решение

*Мы с командой сделали WEB часть и подгрузили ии.*

Работает это так — запускается .bat/.sh файл, он через installer скачивает все нужные файлы (Ollama, python, qwen3) и запускает api, который в свою очередь запускает сайт и саму модель ИИ. ИИ задан был промпт который заставляет ее быть литературным критиком. База данных была подгруженна самостоятельно, поскольку скрипты не могли скачать с сайтов книги. Все работает локально, интернет для работы нейросети **не нужен.**

 **Интернет нужен лишь в том случае если вы нуждаетесь в загрузке python и библиотек.**

## Загрузка своих книг

На сайте `http://localhost:8000/static/index.html` есть чекбокс, чтобы загрузить книги в .txt формате.
